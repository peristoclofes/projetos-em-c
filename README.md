# Projetos em C

Este repositório contém uma coleção de projetos simples desenvolvidos em C.

## Projetos

### 1. Loja de Artesanato (`loja_artesanato.c`)

Este programa gera um relatório diário de vendas para uma pequena loja de artesanato. Ele calcula métricas de vendas com base no número de clientes e nos valores vendidos de produtos de madeira e cerâmica.

**Como usar:**

1.  Compile o programa: `gcc loja_artesanato.c -o loja`
2.  Execute o programa: `./loja`
3.  Insira o número de clientes e os valores de venda conforme solicitado.

### 2. Matrícula de Alunos (`matricula.c`)

Este programa permite o cadastro de disciplinas e o número de alunos matriculados em cada uma. Ao final, exibe um resumo com a lista de disciplinas e o total de alunos.

**Como usar:**

1.  Compile o programa: `gcc matricula.c -o matricula`
2.  Execute o programa: `./matricula`
3.  Insira os nomes das disciplinas (digite "fim" para parar).
4.  Insira o número de alunos para cada disciplina.

### 3. Tabuada (`while.c`)

Este é um programa simples que exibe a tabuada de um número fornecido pelo usuário.

**Como usar:**

1.  Compile o programa: `gcc while.c -o tabuada`
2.  Execute o programa: `./tabuada`
3.  Insira o número para o qual você deseja ver a tabuada.
